# QZES YALNG

> Yet another Leaving-Notes Generator.
> 
> The easiest way to generate leaving notes with arbitrary content!

## Features

- [x] Basic
  - [x] Custom name
  - [x] Custom class
  - [x] Custom dormitory

- [x] Advanced
  - [x] Custom header
  - [x] Hide/Show Commas
  - [x] Hide/Show Logo
  - [ ] Hide/Show Signet
  - [x] Hide/Show Dates

## Dev

```bash
pnpm install
pnpm dev
```
